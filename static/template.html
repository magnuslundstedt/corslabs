CORS Labs

Easiest way to understand Cross-Origin-Resource-Sharing (CORS) is to try it out.

On this page you can test all possible combinations of; website domain, script domain, backend domain and an optional CORS header.

See dev console for test result!

Parameters for http://D.corslabs.com/html/X-Y-Z:
  • D - Domain for website
  • X - Domain where Javascript is retrieved
  • Y - Domain for REST calls within .js
  • Z - Optional CORS header for which domain to allow requests. All for *

Source Code: https://github.com/magnuslundstedt/corslabs
